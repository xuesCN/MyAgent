## 📋 支持的数学函数

### 基础运算
- `+` 加法: `2 + 3` → 5
- `-` 减法: `10 - 3` → 7
- `*` 乘法: `4 * 5` → 20
- `/` 除法: `10 / 2` → 5
- `^` 幂运算: `2 ^ 3` → 8
- `%` 余数: `10 % 3` → 1

### 常见函数
- `sqrt(x)` 平方根: `sqrt(16)` → 4
- `abs(x)` 绝对值: `abs(-5)` → 5
- `round(x)` 四舍五入: `round(3.7)` → 4
- `floor(x)` 向下取整: `floor(3.7)` → 3
- `ceil(x)` 向上取整: `ceil(3.2)` → 4

### 三角函数
- `sin(x)` 正弦: `sin(pi/2)` → 1
- `cos(x)` 余弦: `cos(0)` → 1
- `tan(x)` 正切: `tan(pi/4)` → 1
- `asin(x)` 反正弦
- `acos(x)` 反余弦
- `atan(x)` 反正切

### 对数函数
- `log(x)` 自然对数: `log(e)` → 1
- `log10(x)` 常用对数: `log10(100)` → 2
- `exp(x)` 指数: `exp(1)` → 2.718...

### 常数
- `pi` 圆周率: 3.14159...
- `e` 自然数底: 2.71828...

## ⚠️ 常见问题

### Q1: 计算工具没有被调用？
**可能原因**: Agent 可能认为问题不需要计算工具  
**解决**: 用更明确的数学表述，如"计算 2 + 2"而不是"二加二等于多少"

### Q2: 显示 "未知的工具" 错误？
**可能原因**: LangGraph 服务未正确加载工具  
**解决**: 检查是否成功保存了 langGraphService.ts 的修改，重启应用

### Q3: mathjs 依赖未找到？
**可能原因**: npm 包未正确安装  
**解决**: 
```bash
npm install mathjs --legacy-peer-deps
```

### Q4: 计算结果错误？
**可能原因**: 表达式格式问题  
**解决**: 使用标准数学符号（`*` 乘法，`/` 除法，`^` 幂）
